# 💰 Finance Tracker (Flask + ML)

A simple and efficient **Finance Tracker** built using **Flask** that allows users to manage income and expenses, visualize trends, and perform basic machine learning analysis on financial data.

---

## 🚀 Features

* 📊 Track income and expenses
* 📅 View monthly financial trends
* 📈 Interactive dashboard (charts)
* 🤖 Basic ML analysis for financial insights
* 📁 Data stored in CSV file

---

## 🛠️ Tech Stack

* **Backend:** Python, Flask
* **Data Processing:** Pandas
* **Frontend:** HTML (Flask Templates)
* **Data Storage:** CSV

---

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/asadmobashshir/finance-tracker.git
cd finance-tracker
```

### 2. Install dependencies

```bash
pip install flask pandas
```

### 3. Run the app

```bash
python app.py
```

### 4. Open in browser

```
http://127.0.0.1:5000/
```

---

## 📊 How It Works

* User enters financial data (income/expense)
* Data is saved in **finance_data.csv**
* Flask processes and displays:

  * Total income
  * Total expenses
  * Monthly trends
* Dashboard visualizes data using charts

---

## 🧠 ML Concept (Basic)

* Performs simple analysis to identify patterns in spending
* Helps in understanding financial behavior
